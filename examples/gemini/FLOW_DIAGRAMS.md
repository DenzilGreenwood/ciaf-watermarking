## Key Takeaways

1. **Watermarking is fast** (~50-100ms overhead)
2. **Three-tier verification** provides flexibility
3. **Error handling** prevents user-facing failures
4. **Evidence storage** is flexible (JSON/MongoDB/S3)
5. **Batch processing** optimizes throughput
6. **Async processing** reduces user-facing latency
