```
│  User    │─────▶│ Gemini  │─────▶│ CIAF         │─────▶│ Evidence │
│  Prompt  │      │ API     │      │ Watermarking │      │ Storage  │

                       │                    │

```

## Quick Start

### 1. Install Dependencies
```bash
pip install ciaf-watermarks google-generativeai
```

### 2. Set API Key
```bash
export GEMINI_API_KEY="your-gemini-api-key"
```

### 3. Run Basic Example
```python
import google.generativeai as genai
from ciaf_watermarks import watermark_ai_output
import os

# Configure Gemini
genai.configure(api_key=os.environ["GEMINI_API_KEY"])
model = genai.GenerativeModel('gemini-1.5-flash')  # Latest recommended model

# Generate
response = model.generate_content("Explain quantum computing in 2 paragraphs")

# Watermark
evidence, watermarked = watermark_ai_output(
    artifact=response.text,
    model_id="gemini-1.5-flash",
    model_version="1.5",
    actor_id="user:test",
    prompt="Explain quantum computing in 2 paragraphs"
)

# View result
print("=== WATERMARKED OUTPUT ===")
print(watermarked)
print("\n=== EVIDENCE ===")
print(f"Artifact ID: {evidence.artifact_id}")
print(f"Watermark ID: {evidence.watermark_id}")
```

## Key Features

### ✅ Automatic Type Detection
No need to specify if it's text, image, or PDF - the system auto-detects.

### ✅ Forensic Evidence
SHA-256 hashes + DNA fragments enable detection of content tampering.

### ✅ Hierarchical Verification
Three-tier verification system (exact match → fragments → similarity).

### ✅ Minimal Overhead
Watermarking adds ~50-100ms latency (negligible vs Gemini API call time).

### ✅ Flexible Storage
JSON files, MongoDB, or cloud storage - your choice.

### ✅ Production Ready
Error handling, retry logic, async support built-in.

## Common Questions

### Q: Do I need to modify Gemini API calls?
**A:** No! Just wrap the output with `watermark_ai_output()`. Your Gemini code stays the same.

### Q: Will users see the watermark?
**A:** Yes, by default a footer is added. You can customize or make it invisible.

### Q: What if someone removes the visible watermark?
**A:** Forensic fragments (DNA) still detect authenticity when verified.

### Q: How do I verify content later?
**A:** Use `verify_text_artifact(suspect_text, evidence)` with stored evidence.

### Q: Does this work with all Gemini models?
**A:** Yes! Works with gemini-1.5-flash, gemini-1.5-pro, and any text-generating Gemini model.

### Q: What about images from Gemini?
**A:** If Gemini returns image bytes, watermarking auto-detects and applies image watermarks.

### Q: Can I batch process multiple generations?
**A:** Yes! Process N Gemini outputs in parallel with watermarking.
